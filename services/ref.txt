ref даёт возможность получить доступ к DOM-узлам или Реакт-элементам, созданым компонентом
-Управление фокусом, выделение текста и тд
-Интеграция со сторонними библиотеками
-Вызов анимаций

Перенаправление ref
const Input = forwardRef( --> обертка компонента
(
    { ...props }: InputProps,
    ref: ForwardedRef<HTMLInputElement>) --> объект Реф
) : JSX.Element => {
    return <input ref={ref} {...props} /> --> присвоение Реф
})
У нас есть такая проблема что мы не можем просто указать Реф который будет передан на компонент ниже, для это нужно использовать фцию forwardRef, который возьмет переданный Реф на верхний компонент и затем мы его должны указать внизу конкретно в нашем инпуте куда мы передаем этот реф. По-этому если мы попытаемся на Инпуте сделать фокус без это проброса, а Input - это фциональный Компонент, то мы этого сделать не сможем. Но на записи выше мы форвардим наш Реф, на конкретный референс элемента внутри нашего компонента
